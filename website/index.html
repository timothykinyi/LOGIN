API Documentation

Integrate eID into your application using our secure API. Whether you're building for web, mobile, or desktop, we've got you covered!

Endpoints

Authentication API

POST: /api/auth/login - Authenticate users using eID.

curl -X POST https://api.eid.com/auth/login -d '{"email": "user@example.com", "password": "secure_password"}'

User Data API

GET: /api/user/data - Retrieve user data after authentication.

curl -H "Authorization: Bearer {token}" https://api.eid.com/user/data
